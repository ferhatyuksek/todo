# Todo List Uygulaması ✅

Bu proje,basit bir yapılacaklar listesi (Todo) uygulamasıdır.  
Tüm veriler tarayıcıda **localStorage** ile saklanır, sayfa yenilense bile kaybolmaz.

## Özellikler
- Todo ekleme
- Todo silme
- Tüm todoları temizleme
- Aynı todo’nun tekrar eklenmesini engelleme
- Todo arama / filtreleme
- Verilerin localStorage’da saklanması
